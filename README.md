# Digital Clock with GUI
## Description
This is a simple Digital Clock application developed using Java Swing GUI.
## Features
- Real-Time Clock
- GUI Interface
- Automatic Time Update Every Second
## Technologies Used
- Java
- Swing
- AWT
## How to Run
1. Compile:
   javac DigitalClock.java
2. Run:
   java DigitalClock
